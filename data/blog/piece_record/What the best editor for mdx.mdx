**Answer is Vscode**,to make it more powerful,you can install some extensions below:
+ MDX  Offer syntax highlighting and error notice for MDX files
+ Markdown Preview Enhanced  A powerful markdown previewer that supports MDX
+ Prettier - Code formatter  Automatically format your MDX files
+ ES7+ React/Redux/React-Native snippets Provides useful snippets notice for JSX

## Usage
### Markdown Preview Enhanced
After download **Markdown Preview Enhanced**,to enable preview,you can first press `Ctrl + Shift + P` then find and press **Open Preview to the Side**.
